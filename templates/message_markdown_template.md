# 📢 消息通知Markdown模板

## 🚀 状态通知模板

### 基础状态通知
```markdown
## 🚀 Crawlo-Status | {任务名称}
{简要描述}

📊 **统计信息**：
- 处理数量：{数量}条
- 耗时：{时间}
- 状态：✅ 成功 / ⚠️ 警告 / ❌ 失败
```

### 启动通知
```markdown
## 🚀 Crawlo-Start | {爬虫名称}启动
开始执行数据抓取任务

📍 **任务信息**：
- 目标网站：{网站名称}
- 预计页面：{页面数量}页
- 启动时间：{时间}
```

### 完成通知
```markdown
## ✅ Crawlo-Done | {任务名称}完成
数据抓取任务执行完毕

📈 **执行结果**：
- 成功抓取：{成功数量}条
- 失败请求：{失败数量}次
- 总耗时：{总时间}
- 平均速度：{速度}条/分钟
```

## 📝 模板使用说明

### 变量替换规则
- `{变量名}`：需要替换的具体内容
- `✅/⚠️/❌`：状态标识emoji
- `## 标题`：消息标题格式
- `📊/📋/⏰`：内容分类emoji

### 最佳实践
1. **简洁明了**：突出核心信息，避免冗余
2. **结构清晰**：使用统一的标题和分隔格式
3. **及时准确**：确保时间和数据的准确性
4. **行动导向**：提供明确的后续行动建议

### 适配建议
- **钉钉**：支持Markdown格式，可使用emoji和标题
- **飞书**：支持富文本格式，可添加链接和图片
- **企业微信**：支持基础Markdown，建议简化格式
- **邮件**：可使用HTML格式，支持更丰富的样式

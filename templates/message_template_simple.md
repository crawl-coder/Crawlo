# 📱 消息通知模板（简洁版）

## 🚀 任务通知

### 启动通知
```markdown
🚀 {任务名称} 开始执行
▶️ 目标：{网站/服务名称}
⏱️ 预计时长：{时间}
```

### 完成通知
```markdown
✅ {任务名称} 执行完成
📊 结果：成功抓取 {数量} 条数据
⏱️ 耗时：{耗时}
```

### 进度通知
```markdown
📊 {任务名称} 执行进度
📈 已完成：{进度百分比}%
🔢 当前数量：{当前数量} 条
```

## ⚠️ 异常通知

### 错误告警
```markdown
🚨 {任务名称} 执行异常
❌ 错误：{错误描述}
⏰ 时间：{发生时间}
```

### 性能告警
```markdown
⚠️ 系统性能异常
📉 {指标名称}：{当前值} (阈值：{阈值})
```

## 📈 统计报告

### 日报
```markdown
📊 {日期} 数据统计
📈 新增：{新增数量} 条
🔢 总量：{总数量} 条
🎯 成功率：{成功率}%
```

### 周报
```markdown
📅 {周期} 统计报告
📊 总抓取：{总数量} 条
📈 日均：{日均数量} 条
✅ 成功率：{成功率}%
```

## 🛠️ 系统通知

### 配置变更
```markdown
🔧 配置更新
📝 {配置项}：{原值} → {新值}
⏰ 生效时间：{时间}
```

### 系统维护
```markdown
🛠️ 系统维护通知
📅 维护时间：{时间}
⚠️ 影响范围：{影响说明}
```

## 使用示例

### 实际应用示例
```markdown
# 爬虫任务通知示例

## 启动通知
🚀 ofweek爬虫 开始执行
▶️ 目标：OFweek电子工程网
⏱️ 预计时长：5-10分钟

## 完成通知  
✅ ofweek爬虫 执行完成
📊 结果：成功抓取 156 条数据
⏱️ 耗时：7分23秒

## 进度通知
📊 ofweek爬虫 执行进度
📈 已完成：65%
🔢 当前数量：102 条

## 错误告警
🚨 ofweek爬虫 执行异常
❌ 错误：连接超时，请检查网络
⏰ 时间：2024-01-15 14:32:15
```

## 📝 模板变量说明

| 变量 | 说明 | 示例 |
|------|------|------|
| `{任务名称}` | 任务或爬虫名称 | ofweek爬虫 |
| `{数量}` | 数据条数 | 156 |
| `{时间}` | 时间信息 | 7分23秒 |
| `{百分比}` | 完成百分比 | 65% |
| `{错误描述}` | 错误详情 | 连接超时 |
| `{指标名称}` | 性能指标 | 响应时间 |
| `{当前值}` | 当前数值 | 1.2s |
| `{阈值}` | 预设阈值 | 1.0s |

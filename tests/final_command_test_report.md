## 3. 命令功能总结

### 3.1 help 命令
显示框架帮助信息，包括所有可用命令和用法说明。

### 3.2 version 命令
显示框架版本信息。

### 3.3 startproject 命令
创建新的 Crawlo 项目，生成标准目录结构和配置文件：
- 项目根目录和配置文件 (crawlo.cfg)
- 设置文件 (settings.py)
- 爬虫目录和初始化文件 (spiders/__init__.py)
- 项目包结构
- 运行脚本 (run.py)

### 3.4 genspider 命令
生成爬虫模板文件：
- 基于域名生成爬虫类
- 自动创建起始URL列表
- 包含解析方法模板
- 支持自定义项目项类

### 3.5 run 命令
运行指定爬虫或所有爬虫：
- 支持单个爬虫运行
- 支持批量运行所有爬虫
- 提供JSON输出选项
- 支持禁用统计信息

### 3.6 list 命令
列出项目中所有可用爬虫：
- 显示爬虫名称、类名、模块信息
- 显示起始URL数量和允许域名
- 支持JSON格式输出

### 3.7 check 命令
检查爬虫代码合规性：
- 验证爬虫类定义
- 检查必要属性和方法
- 提供修复建议
- 支持CI集成模式

### 3.8 stats 命令
查看爬虫运行统计信息：
- 显示历史运行记录
- 展示详细统计指标
- 支持特定爬虫查询
